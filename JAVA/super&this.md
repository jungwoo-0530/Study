# Super



## Super vs this

#### this

- this : 멤버 필드와 일반 변수(매개 변수)를 구분할 때 사용. 
- this() : 한 생성자가 다른 생성자를 호출할 때 사용.
- this는 같은 인스턴스(클래스) 안에서 활용.

------------------------------------------------------------------

	#### Super

- 부모클래스의 생성자 호출을 위해 사용. 

- 자손클래스의 생성자에서는 부모클래스의 생성자를  호출해서 먼저 처리되도록 해야 함.(첫번째 줄에 작성) 

- 부모클래스에 여러 개의 생성자가 존재할 경우(오버로딩) 생성자의 매개변수로 구분하여 사용.
